# Tumour Study Portal

## Deploy locally

```sh
# 1. clone repo on your laptop
git clone https://github.com/Copper3D-brids/tumour-position-calculation.git
# 2. open volume veracrypt and mount you data on I:
# 3. start docker
# 4. open the project
# 5. double-click `run_app_docker.bat` for windows users
# 6. for linux, macos users
cd tumour-position-calculation
docker compose up
```
